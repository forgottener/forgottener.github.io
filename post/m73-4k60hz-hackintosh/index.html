M73 Tiny介绍及硬件参数

m73-tiny

我选用的cpu是i5-4690T，黑起来没遇到什么问题。这个机器不挑内存的，只要是ddr3笔记本内存就可以了，亲测低压1.35V、标压1.5V的都能用。

Thinkcentre-m73-hackintosh-OPENCORE-EFI

主要在这个efi的基础上加以修改

CPU核显HD4600开启4K 60Hz

  • framebuffer-patch-enable = 01000000
  • framebuffer-stolenmem = 00000004 (64m)
  • framebuffer-fbmem = 00000003 (48m)
  • framebuffer-unifiedmem = 00000080 (提升显存到2G)
  • 在 boot-args 增加 -cdfon

展示opencore GUI以及recovery选项

  • Misc-boot-ShowPicker：勾选此项为true，则此项变为默认项，强制picker显示，可以在引导时使用Option
  • Misc-boot-HideAuxiliary：勾选此项为true，隐藏如：recovery、Clean NVRAM等，按ESC，0，空格后，显示而不隐藏。这里我设为了false用来展示recovery

Reset NVRAM不要乱点否则需要重刷主板BIOS

不要问为什么不要乱点～～

如何刷bios？

联想官网m73 tiny bios下载地址

截止目前（2021年02月26日）笔者下载的版本是FHKT85A，下载文件名是fhjt85usa.zip

下载bios文件，用 rufes 刷入u盘后进dos刷，命令很简单

cd m73bios

flash2 IMAGEFH.rom

# 确认按y

如果想要解锁硬件白名单，顺便在刷bios的时候就可以，很简单把机器的sn、mtm都刷成INVALID

flash2 IMAGEFH.rom /sn:INVALID /mtm:INVALID
