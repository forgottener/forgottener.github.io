Unraid下Docker部署Nextcloud网盘

2021-02-24
unraid nas

文章目录

    • 安装步骤
    • 遇到的问题
    • 修改配置文件

安装步骤

  • Community Applications 搜索nextcloud
  • 选择 linuxserver/nextcloud
  • Host Port 1:* Container Port: 443容器内443端口对应宿主机unraid的端口。随便挑一个没被占用的，我用了1443
  • Host Path 2:* Container Path: /data填unraid序列里创建的一个用来存储nextcloud数据的共享文件夹，我的是/mnt/user/nextcloud
  • APPLY启动，访问https://xx.xx.xx.xx:1443进入安装网页界面。（xx.xx.xx.xx是unraid的ip，必须是https,浏览器提示不安全选择继续访问）
  • 顺利进入安装网页界面后设置管理员账号、密码，选择数据库类型后完成安装。

遇到的问题

第一个trusted_domains的问题比较好定位，修改配置文件/mnt/user/appdata/nextcloud/www/nextcloud/config即可。

第二个问题：由于nextcloud服务对于默认的域名(https默认443)是不带端口的，但是你懂的，通过ddns公网转发到路由器端口映射局域网内的nextcloud服务肯定会带端口号，这在客户端上访问就会遇到跳转问题（链接跳转时端口号丢了）
